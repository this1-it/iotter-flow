# Documentation

This repository keeps active documentation only for the current Vaadin Flow codebase.

- Start with [index.md](index.md) for the status map and canonical entry point.
- Use `implementation/` for current architecture, UI patterns, and troubleshooting notes tied to running code.
- Treat `archive/` as historical migration material from Vaadin 8 or intermediate migration phases. It is not the source of truth for current development.
